= tsqlfmt

`tsqlfmt` is a command-line formatter for T-SQL.
It uses Microsoft's SQL ScriptDOM to parse SQL, then renders formatted output from the parsed syntax tree.

The formatter reads SQL Prompt-style JSON configuration files.
It also supports project-specific options under `formatterExtensions`.

The main profile in this repository is `default-style.json`.

== Usage

[source,shell]
----
dotnet run --project src/TSqlFormatter -- [OPTIONS] [FILE]
----

When `FILE` is omitted or `-`, the formatter reads from standard input and writes formatted SQL to standard output.
